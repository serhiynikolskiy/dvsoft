Название проекта dvsoft.
Проект является веб приложением, написанным на языке PHP с подключением базы данных MySQL. 

Для запуска проекта необходимо склонировать проект на локальную машину с помощью команды 
git clone https://github.com/serhiynikolskiy/dvsoft.git

Далее для запуска Вам необходимо будет подключить базу данных. Для этого можно воспользоваться дампом, подготовленным нами, который лежит в корне проекта /dvsoft.sql

Для подключения Вашей базы данных необходимо в файлике /core/db.php указать подключение к Вашей базе данных.

После этого если у Вас включен и настроен виртуальный хост, вы можете перейти по настроенному URL на сайт локально.

Настройка локальных хостов детально описана здесь:
https://www.digitalocean.com/community/tutorials/apache-ubuntu-14-04-lts-ru
